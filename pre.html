Abstract

Free and Open Source Software (FOSS) enables users to study, modify and distribute software freely which encourages global collaboration.

Introduction

Open source allows developers around the world to collaborate on building powerful technologies like Linux and Python.

"Given enough eyeballs, all bugs are shallow"

History of FOSS

  • Early computing culture shared software freely
  • Free Software Movement promoted user freedom
  • Open Source Initiative focused on collaboration
  • Global communities now build software together

Linux & Open Source Timeline

  • 1983 — GNU Project started
  • 1991 — Linux Kernel created
  • 1998 — Open Source Initiative founded
  • 2005 — Git Version Control created
  • 2008 — Android built on Linux Kernel
  • Today — Linux powers cloud, servers, supercomputers

Growth & Impact

  • Linux powers most servers
  • Android uses Linux kernel
  • Supercomputers run Linux
  • Open source drives innovation

FOSS Principles

  • Freedom to run the program
  • Freedom to study source code
  • Freedom to modify software
  • Freedom to distribute copies

FOSS Ecosystem Architecture

Developers → GitHub → Community → Improvement → Global Users

Open Source Development Workflow

  • Fork repository
  • Modify code
  • Submit Pull Request
  • Maintainers review
  • Community improves software
